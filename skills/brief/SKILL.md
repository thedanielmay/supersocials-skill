# SuperSocials: Brief

Think like a creative director. Produce briefs specific enough that a designer or image AI can execute without asking follow-up questions.

## What This Module Produces

Asset-by-asset creative briefs covering:

- **Mood** — 3-word direction, then expanded
- **Colour** — palette direction and avoid list
- **Composition** — framing, focal point, negative space
- **References** — visual styles, accounts, aesthetics to draw from
- **Do / Don't** — explicit guardrails

## Process

### Step 1: Establish context

If invoked directly, ask:
- What is the asset? (Reel thumbnail, carousel cover, Instagram post, LinkedIn header, etc.)
- What platform is it for?
- What's the post about?
- What's the brand's visual personality? (ask for 3 words or a reference account)

### Step 2: Write the brief

One brief per asset. Be specific and opinionated — vague briefs produce vague results.

**Output format:**

```
--- BRIEF 1 — INSTAGRAM REEL THUMBNAIL ---

MOOD
Three words: Bold · Minimal · Confident
The image should feel like a confident statement, not a question. Clean, direct, nothing unnecessary.

COLOUR
Primary: Deep navy (#0A1628) or brand equivalent
Accent: White text only — no colour accents on this asset
Avoid: Gradients, drop shadows, neon, pastels

COMPOSITION
- Subject centred or rule-of-thirds left
- Large, legible text overlay (80% of the frame can be text if needed)
- Negative space on the right third for text breathing room
- No busy backgrounds — solid colour or blurred/darkened photo

REFERENCES
Accounts: @alexhormozi, @hubspot — the high-contrast, text-heavy thumbnail style
Style: Bold typography over clean background, not lifestyle photography

DO
- Use one strong visual statement
- Make the text readable at thumbnail size (test at 100px wide)
- If using a face, direct eye contact with camera

DON'T
- Don't use stock photography
- Don't use more than 2 typefaces
- Don't add a logo — save it for the caption
---
```

End with: "Want me to write the caption copy for this post as well?"
