# SuperSocials: Script

Think like a video producer. Scripts must work spoken aloud, not read on a page.

## What This Module Produces

Full video scripts with:
- `[HOOK]` — first 2–3 seconds, pattern interrupt
- `[BODY]` — core content, structured for retention
- `[CTA]` — single clear action
- `[B-ROLL]` — shot suggestions and on-screen text notes

## Platform Script Formats

### TikTok / Instagram Reels (15–90 seconds)

Hook in the first 2 seconds or you lose them. No intro. No "hey guys". Get to the point.

Structure:
```
[HOOK]     2–3 seconds. Pattern interrupt. Raise a question or make a provocative statement.
[BODY]     Main content. 3–5 punchy beats. One idea per beat. No filler.
[CTA]      5 seconds max. One action. "Follow for more." / "Comment X if you want the full guide."
[B-ROLL]   Shot notes and on-screen text to cut to. Be specific.
```

### YouTube Shorts (under 60 seconds)

Same as TikTok. Hook → rapid value → CTA. Optimise for replays — end with a cliffhanger or loop.

### YouTube Long-Form (8–20 minutes)

Retention structure:
```
[HOOK]       First 30 seconds. State the payoff. Why should they watch to the end?
[INTRO]      30–60 seconds. Establish credibility. Preview what's coming.
[CHAPTER 1]  First main point. One idea, one story, one proof.
[CHAPTER 2]  Second main point. Build on Chapter 1.
[CHAPTER 3]  Third main point. The climax or the turn.
[RECAP]      30 seconds. Summarise the three points.
[CTA]        Subscribe, comment, watch next. One action only.
[B-ROLL]     Shot list and on-screen text notes for each chapter.
```

## Process

### Step 1: Establish context

If invoked directly, ask:
- Platform and target length?
- Topic?
- Tone? (energetic, calm, authoritative, conversational)
- Will you be on camera or voiceover only?

### Step 2: Write the script

Write in spoken-word language. Short sentences. Natural pauses. Read it aloud in your head — if it sounds like an essay, rewrite it.

Avoid:
- Long sentences (over 15 words)
- Jargon the audience wouldn't use themselves
- Passive voice
- Filler openers ("In today's video...", "Hey guys, welcome back...")

### Step 3: Humanise (announce before doing)

> "Running `humanize` on this script so it sounds like a real person talking, not copy being read aloud."

Invoke the `humanize` skill. Apply the output.

### Step 4: Deliver

Output format:

```
--- SCRIPT 1 — TIKTOK (approx. 45 sec) ---

[HOOK]
"Most people waste the first 10 seconds of every video. Here's what to do instead."

[BODY]
Beat 1: The mistake — cutting to your face before anything interesting happens.
Beat 2: The fix — open on a result, a question, or a visual surprise.
Beat 3: The payoff — viewers who see something unexpected in the first frame watch 3x longer.

[CTA]
"Follow — I post one of these every day."

[B-ROLL]
- Open on: screen recording of a TikTok analytics graph spiking
- Cut to: talking head at beat 2
- On-screen text: "The fix:" overlaid at beat 2
---
```

End with: "Want me to write captions to go with this, or a visual brief for the shoot?"
